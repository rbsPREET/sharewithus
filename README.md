# ShareWithUs
Client:
React

Server:
Node.js

Database:
MongoDB

## Installation
Open Terminal and cd to /sharewithus folder
```bash
npm i
```
First Terminal
```bash
cd /client -> npm start
```
Second Terminal
```bash
cd /server -> npm start
```
## Create .env file at /server folder 
```env
PORT=5000
MONGO_URL=<Go to https://www.mongodb.com/cloud to create a new project and paste the database URL here>
JWT_SECRET=<replace>
 ```
